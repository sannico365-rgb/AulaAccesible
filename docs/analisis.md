# Análisis de requisitos — AulaAccesible

AulaAccesible es una plataforma académica web accesible diseñada para estudiantes y docentes con diversas capacidades visuales, cognitivas y motoras.

## Roles de usuario

Estudiante con baja visión  
Estudiante con dislexia  
Profesor con limitación motora leve  
Usuario con conexión lenta  

## Historias de usuario

Como estudiante con baja visión quiero navegar con teclado y alto contraste.  

Como estudiante con dislexia quiero textos claros y tipografía legible.  

Como profesor quiero botones grandes para gestionar tareas.  

Como usuario con conexión lenta quiero páginas ligeras.  

## Priorización MoSCoW

Must: cursos, tareas, envío, accesibilidad  
Should: perfil, breadcrumbs  
Could: preferencias  
Won’t: app móvil
